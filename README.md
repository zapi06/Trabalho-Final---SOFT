# Trabalho-Final---SOFT
Repositório destinado ao Trabalho Final da disciplina Engenharia de Software
---

#### Autores:
Lucas dos Santos Menezes

Vinícius Persike de Souza

Vinícius Zapelini Mangrich

---

O trabalho proposto realiza a implementação de um sistema bancário capaz de cadastrar novos clientes, abrir conta corrente ou conta poupança e realizar algumas operações básicas, como depósitos, saques e transferências. Há também a criação de outras classes como empréstimos, cadastro de pessoa jurídica, entre outros, que não foram integradas no sistema, mas estão presentes como base do código.

Além da implementação do "esqueleto" do sistema, há a criação dos testes unitários, como requisitado na proposta do trabalho.
